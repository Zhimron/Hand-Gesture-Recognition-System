# AGENTS.md

## Project Name
Hand Gesture Recognition System

## Overview
This project is developed incrementally. Each task should implement only the requested feature.

Never redesign the project unless instructed.

## Rules

- Only work on the requested feature.
- If the feature already exists:
  - Verify it.
  - Improve it if necessary.
  - Otherwise skip it.
- Never delete working code.
- Preserve backwards compatibility.
- Follow clean architecture.
- Keep files modular.
- Avoid duplicated code.
- Add type hints.
- Add docstrings.
- Add comments only when necessary.
- Create tests whenever possible.

## Coding Standards

- Python 3.12+
- PEP8
- Black formatting
- Ruff compatible
- Modular architecture
- Dependency injection when appropriate
